# Ella's Valentine Run

A browser-based endless runner game built with Phaser 3.

## Running the Game

To run the game locally:

1. Serve the project directory using a simple HTTP server (e.g., Python `http.server` or `live-server`):
   ```bash
   python3 -m http.server 8000
   ```
2. Open [http://localhost:8000](http://localhost:8000) in your browser.

## Running Tests

This project uses Jest for unit testing.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run tests:
   ```bash
   npm test
   ```

3. Watch mode:
   ```bash
   npm test -- --watch
   ```
